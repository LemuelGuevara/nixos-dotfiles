# Nixos + Hyprland

NixOS dotfiles that contains shared dotfiles and hyprland dotfiles via submodules.

## Config Setup

- **Terminal:** ghostty
- **Shell:** zsh with starship
- **Window Manager (WM)**: hyprland
- **Text-editor:** neovim
- **Multiplexer:** tmux
- **Video Player:** mpv
- **Browser:** vivaldi

### Some Notable Terminal User Interfaces (TUIs)

- **bluetui:** bluetooth manager that works like blueman but better
- **btop:** lets you monitor your system usage
- **disktui:** disk manager that allows you to easily mount external drives
- **dysk:** allows you to view the storage usage of your drives
- **lazygit**: git management in the terminal with ease
- **nvitop**: allows you to see the usage of your nvidia graphics card
- **wiremix:** basically pavucontrol but in the terminal and looks very nice
- **yazi:** file broswer in the terminal (really amazing)
